About This Project

Purpose

This project is a machine learning-powered web application designed to classify SMS and email messages as either "Spam" or "Not Spam" (often called "Ham"). It demonstrates a complete end-to-end data science workflow, from raw data processing to a deployed, interactive user interface.

Technology Stack

  • Backend: Python with Flask for the web server.
  • Machine Learning: Scikit-learn for model training and evaluation, specifically using a Multinomial Naive Bayes classifier.
  • Natural Language Processing (NLP): NLTK (Natural Language Toolkit) for text preprocessing, including tokenization, stop word removal, and stemming.
  • Feature Extraction: TF-IDF (Term Frequency-Inverse Document Frequency) to convert text messages into numerical vectors for the model.
  • Frontend: HTML and Tailwind CSS for a modern, responsive, and clean user interface.
  • Data Source: The model was trained on the "SMS Spam Collection Dataset" from Kaggle.
